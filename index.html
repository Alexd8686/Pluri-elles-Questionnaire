Oui — avec GitHub, c’est assez simple.

Le bon choix pour ton besoin est GitHub Pages : c’est un hébergement de site statique qui publie directement des fichiers HTML, CSS et JavaScript depuis un dépôt GitHub. GitHub indique aussi que, sur GitHub Free, GitHub Pages est disponible pour les dépôts publics. GitHub Docs+1

Voici la méthode la plus simple.

1. Créer un dépôt GitHub

Dans GitHub :

  • clique sur New repository

  • donne-lui un nom, par exemple questionnaire-plurielles

  • mets-le en Public

  • coche Add README

  • crée le dépôt

GitHub recommande soit un dépôt dédié, soit un dépôt nommé username.github.io pour un site principal. Pour ton cas, un dépôt classique questionnaire-plurielles suffit très bien. GitHub Docs+1

2. Ajouter ton fichier HTML

Dans le dépôt :

  • clique sur Add file puis Upload files

  • envoie le fichier questionnaire_harcelement_plurielles_premium_v2.html

  • renomme-le en index.html

Le nom index.html est important : c’est la page qui s’ouvrira automatiquement sur ton lien GitHub Pages. La documentation GitHub Pages montre bien ce fonctionnement avec un fichier index.html. GitHub Pages

3. Activer GitHub Pages

Ensuite :

  • va dans Settings

  • puis Pages

  • dans Build and deployment ou Source, choisis :

    • Deploy from a branch

    • branche main

    • dossier /root

  • enregistre

GitHub permet de publier un site Pages depuis une branche et un dossier choisis comme source de publication. GitHub Docs+1

4. Récupérer ton lien

https://ton-identifiant-github.github.io/questionnaire-plurielles/
